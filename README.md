## 使用方法

1. 打开 浏览器 → 【更多工具】 → 【扩展程序】→ 【加载已解压的扩展程序】→ 选择下载好的源码文件，打开 dist 文件夹

2. 打开 浏览器 → 【更多工具】 → 【扩展程序】→ 拖拽 crx 文件到页面空白位置 → 在弹出框选择 【添加扩展程序】
